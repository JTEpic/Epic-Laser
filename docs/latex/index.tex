\begin{DoxyItemize}
\item RB6 (RP6 for single pulse OC) for the x motor stepping
\item RB7 (RP7 for single pulse OC) for the y motor stepping
\item RB8 for the x motor direction
\item RB9 for the y motor direction
\item RB10 for enabling both motors, active low
\item RB5 for the laser
